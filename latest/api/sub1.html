Submodule 1: Numerical Functions#

This module contains optimized functions for numerical operations with NumPy.

Utilities in the sub1.modulo1 module.

This module provides small helper functions used by the package. Docstrings follow the NumPy style and include LaTeX where sensible.

Examples

>>> import numpy as np
>>> from test_pypi_pru.sub1.modulo1 import func1
>>> func1(np.array([1, 2, 3]))
array([1, 4, 9])
test_pypi_pru.sub1.modulo1.func1(data)[source]#

Element-wise square of an array.

Parameters:

data (NDArray) – Input array of numeric values.

Returns:

Array where each element is the square of the corresponding input element. Computed element-wise as $f(x) = x^{2}$.

Return type:

ndarray

Examples

>>> import numpy as np
>>> func1(np.array([1, 2]))
array([1, 4])
test_pypi_pru.sub1.modulo1.func2(value, decimals)[source]#

Square a value and round to a number of decimal places.

Parameters:

  • value (float) – Input numeric value to be squared.

  • decimals (int) – Number of decimal places to round the squared result to.

Returns:

The result of rounding $value^{2}$ to decimals decimal places, i.e. $mathrm{round}(value^{2}, decimals)$.

Return type:

float

Examples

>>> func2(1.2345, 2)
1.52
test_pypi_pru.sub1.modulo1.func3(value)[source]#

Compute the quadratic (value + 1)^2 using its expanded form.

Parameters:

value (float) – Input numeric value.

Returns:

The computed value of $(value + 1)^{2}$, i.e. $value^{2} + 2,value + 1$.

Return type:

float

Notes

The expanded form is used for clarity; numerically it is equivalent to computing (value + 1)**2.

Examples

>>> func3(2.0)
9.0
>>> func3(0.5)
2.25
